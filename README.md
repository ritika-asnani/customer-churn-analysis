# Customer Churn Analysis

A small Python-based project analyzing customer churn data to understand patterns related to customer retention and behavior. The analysis is performed in a Jupyter Notebook and focuses on data cleaning, exploratory analysis, and visualization.

---

## What’s Inside

- **Notebook (`.ipynb`)** — contains the complete analysis, including data cleaning, EDA, visualizations, and observations  
- **Dataset (`.csv`)** — customer churn data used for the analysis  

---

## Analysis Performed

- Data cleaning and preprocessing  
- Univariate and bivariate analysis  
- Churn distribution analysis  
- Impact of contract type, tenure, and monthly charges on churn  
- Visualization of key trends and patterns  

(Important charts and observations are documented directly inside the notebook.)

---

## Tools & Technologies

- **Python** — data analysis  
- **Jupyter Notebook** — interactive analysis environment  
- **Pandas, NumPy** — data processing  
- **Matplotlib, Seaborn** — data visualization
